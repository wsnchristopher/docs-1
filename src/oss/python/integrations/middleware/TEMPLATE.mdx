## Use with an agent

Pass middleware to @[`create_agent`] or @[`create_deep_agent`] in order. You can combine provider middleware with [built-in middleware](/oss/langchain/middleware/built-in).

```python Agent with middleware icon="robot"
from langchain.agents import create_agent

agent = create_agent(
    model="claude-sonnet-4-6",
    tools=[...],
    middleware=[middleware],
)

agent.invoke({"messages": [{"role": "user", "content": "INSERT QUERY"}]})
```

<Tip>
    Middleware runs in the [agent loop](/oss/langchain/middleware/overview#the-agent-loop). For custom behavior, see [custom middleware](/oss/langchain/middleware/custom).
</Tip>
