[Deeplake](https://www.deeplake.ai/) is a database optimized for AI and deep learning
applications.


## Installation and setup

<CodeGroup>
```bash pip
pip install langchain-deeplake
```

```bash uv
uv add langchain-deeplake
```
</CodeGroup>
