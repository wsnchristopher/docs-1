All functionality related to Yandex Cloud

>[Yandex Cloud](https://cloud.yandex.com/en/) is a public cloud platform.

## Installation and setup

Yandex Cloud SDK can be installed via pip from PyPI:

<CodeGroup>
```bash pip
pip install yandexcloud
```

```bash uv
uv add yandexcloud
```
</CodeGroup>

## LLMs

### YandexGPT

See a [usage example](/oss/python/integrations/llms/yandex).

```python
from langchain_community.llms import YandexGPT
```

## Chat models

### YandexGPT

See a [usage example](/oss/python/integrations/chat/yandex).

```python
from langchain_community.chat_models import ChatYandexGPT
```

## Embedding models

### YandexGPT

See a [usage example](/oss/python/integrations/embeddings/yandex).

```python
from langchain_community.embeddings import YandexGPTEmbeddings
```

## Parser

### YandexSTTParser

It transcribes and parses audio files.

`YandexSTTParser` is similar to the `OpenAIWhisperParser`.
See a [usage example with OpenAIWhisperParser](/oss/python/integrations/document_loaders/youtube_audio).

```python
from langchain_community.document_loaders import YandexSTTParser
```
