## Installation and setup

Install the Perplexity x LangChain integration package:

<CodeGroup>
```bash pip
pip install langchain-perplexity
```

```bash uv
uv add langchain-perplexity
```
</CodeGroup>

Get your API key from the [Perplexity getting started guide](https://docs.perplexity.ai/docs/getting-started).
