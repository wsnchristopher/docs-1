[LangSmith Cloud](/langsmith/cloud) is a **managed platform for deploying your agents**. LangChain hosts and operates the [control plane](/langsmith/control-plane), [data plane](/langsmith/data-plane), [Agent Server](/langsmith/agent-server) runtime, and supporting databases on AWS and GCP. Push code to a connected GitHub repository or invoke the `langgraph deploy` CLI, and the platform handles build, provisioning, scaling, and ongoing operations.

<Callout icon="clipboard-check" color="#4F46E5" iconType="regular">
Agent deployments running on Cloud require a [Plus plan or above](https://www.langchain.com/pricing). Before creating your first agent deployment, verify that your application runs locally with `langgraph dev`. Refer to [Local development and testing](/langsmith/local-dev-testing).
</Callout>
